Feature Testing Guide

Independent verification workflows for core project modules

scene_generation.py AI Logic

Transforms text content into visual scene descriptions for podcast visualization.

Independent Test Script
import utils.scene_generation as sg
text = "The universe is expanding rapidly."
scenes = sg.generate_scenes(text)
print(scenes)

Capture: List of scene objects with visual descriptions.

image_prompt_gen.py AI Logic

Refines raw scene data into descriptive prompts for DALL-E/Midjourney.

Independent Test Script
from utils import image_prompt_generation as ipg
raw = "A futuristic city at night."
refined = ipg.refine_prompt(raw)
print(refined)

Capture: High-detail text prompt string.

chunking.py Data

Handles logical splitting of large documents for LLM context window limits.

Independent Test Script
from utils import chunking
long_text = "..." * 1000
chunks = chunking.split_text(long_text, max_size=500)
print(f"Total chunks: {len(chunks)}")

Capture: Count and sample of text segments.

pdf_handler.py File IO

Core utility for extracting raw data from uploaded PDF documents.

Independent Test Script
from utils import pdf_handler
data = pdf_handler.extract_all("sample.pdf")
print(data.keys())

Capture: Dictionary containing 'text' and 'images'.

retrieval_extraction.py AI Logic

Uses RAG to pick out relevant information from the extracted document data.

Independent Test Script
from utils import retrieval_extraction as re
query = "What is the primary conclusion?"
ans = re.extract_info(query, source_text)
print(ans)

Capture: LLM-generated answer based on context.

models_config.py Config

Validates and provides access to API keys and model parameters.

Independent Test Script
from utils import models_config as config
status = config.test_api_connection()
print(f"API Online: {status}")

Capture: Boolean (True/False).
