MinerU 10-Stage Project Migration

Secure Setup & Deployment Guide for Additional PCs

🚀 Quick Start (New PC)

1. Extract Migration Package

Unzip the MinerU_Pipeline_Full_Package.zip file into your working directory (e.g., e:\mineru_project\).

2. Validate Credentials

Ensure the following file exists in your project root. Do NOT share this file publicly!

neurality-nj-e776c5d11c91.json
⚠️ Without this file, all Step 6 (RAG) through Step 10 (Images) AI functions will fail.

3. Initialize Virtual Environment

Open a terminal (PowerShell/Bash) in the project folder and run:

python -m venv env
.\env\Scripts\activate

4. Install Dependencies

Update your local PIP environment with the required UI and AI frameworks:

pip install -r requirements.txt

5. Run the Application

Launch the 10-Stage Pipeline UI:

python ui_app.py

📑 Troubleshooting

  • MinerU OCR Missing: If Step 3 fails, the mineru_models folder may be missing. The system will attempt a new download, but manually copying the folder is faster.
  • Branding Fail: Ensure the reference-pic/ folder is present. The logo engine expects download.jpg for high-fidelity branding.
  • Module Errors: Re-run Step 4 if you encounter "PyQt5" or "fitz" not found.

✅ YOU ARE NOW READY FOR PRODUCTION!
